*Based on the given purchase document number to display the purchase document numbers, document dates and vendor numbers by using classical report events,
validate the input and also provide the top of page as : these are PO header details AND end-of-page as : SPRAO Technology.


REPORT ZRS_CLASICAL_REPORT_01 LINE-COUNT 10(1) NO STANDARD PAGE HEADING.


TABLES EKKO.

SELECTION-SCREEN BEGIN OF BLOCK A WITH FRAME.
  SELECT-OPTIONS S_EBELN FOR EKKO-EBELN.
SELECTION-SCREEN END OF BLOCK A.

"VALIDATION.

DATA V1 TYPE EKKO-EBELN.

types : BEGIN OF TY_EKKO,
        EBELN TYPE EKKO-EBELN,
        BEDAT TYPE EKKO-BEDAT,
        LIFNR TYPE EKKO-LIFNR,
        END OF TY_EKKO.

DATA : WA_EKKO TYPE TY_EKKO,
       IT_EKKO TYPE TABLE OF  TY_EKKO.

  AT SELECTION-SCREEN.
    SELECT SINGLE EBELN FROM EKKO INTO V1 WHERE EBELN IN S_EBELN.

  IF SY-SUBRC <> 0.
    MESSAGE E000(ZRS_MESSAGE_02) WITH 'INVALID PURCHASE DOCUMENT'.

  ENDIF.

START-OF-SELECTION.
  SELECT EBELN BEDAT LIFNR FROM EKKO INTO TABLE IT_EKKO WHERE EBELN IN S_EBELN.

    END-OF-SELECTION .

    LOOP AT IT_EKKO INTO WA_EKKO.
      WRITE :/ WA_EKKO-EBELN,
               WA_EKKO-BEDAT,
               WA_EKKO-LIFNR.
      ENDLOOP.

TOP-OF-PAGE.
      WRITE 'THESE ARE PO HEADER DETAILS'.

END-OF-PAGE.
      WRITE 'SPRAO TECHNOLOGY'.
